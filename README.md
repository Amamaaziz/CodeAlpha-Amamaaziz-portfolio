# 🚀 Modern Material 3 Portfolio Website

A sleek, responsive personal portfolio designed to showcase software development milestones, academic accomplishments, and technical project components.

## 🌐 Live Demo
Check out the live deployment here: **[portfolio-amama-aziz.vercel.app](https://portfolio-amama-aziz.vercel.app)**

## 🛠️ Built With
* **HTML5 & CSS3** - Advanced responsive semantic structures and layout matrices.
* **JavaScript (Vanilla)** - Interactive modal popups and system tracking utilities.
* **HTML5 Canvas** - Dynamic, interactive background node particle network engine.

## ✨ Core Features
* **Material 3 Token Architecture:** Fully customized surface tonal elevation layers.
* **Dynamic Timelines:** Symmetrical charting layouts logging academic paths and verified certificates.
* **Scroll-Reveal Animations:** Automated viewport intersection observers for fluid layout transitions.
* **Mobile Optimized:** Flexible breakdown breakpoints ensuring cross-device stability.
